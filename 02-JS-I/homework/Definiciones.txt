Variables: Las variables son contenedores de informaci√≥n, a los cuales les asignamos un nombre. Podemos tener dentro de una variable textos, numeros, formulas o funciones. existen varias maneras de escribir las variables, las cuales son: var nombre (Megan), let nombre (Jesus), const PI (3.1416)
Strings: Es un tipo de dato, son textos, y para que la consola lo interprete como tal, es muy importante encerrar estos textos entre comillas "", las strings pueden ser letras, frases incluso numeros. 
Funciones: Permiten almacenar instrucciones o pedazos de codigo para que hagan una tarea en especial, y de esta manera vamos a tener disponible ese fragmento de codigo para reutilizarlo cuando lo necesitemos.
(Argunmentos, return): Las variables que vamos a usar en las funciones son argumentos, y return se utiliza para finalizar una funcion o cuando queremos obtener una respuesta en algun punto especifico.
Declaraciones: Se usan para expresar decisiones en nuestro codigo, por ejemplo decimos que (if) si juan tiene una galleta entonces juan tiene un dulce, si la respuesta fuese negativa colocamos el condicional (else) para dar una respuesta, por ejemplo juan no tiene dulces.
Valores Booleanos: Asi como las strings, tambien es un tipo de dato, el cual solo contiene dos valores (true, false), segun lo que planteemos en la consola, la respuesta sera true (verdad) o false (falso)
